## 研究方向
目前专注于研究与解决确定性网络技术理论与应用过程中的关键问题。

### 工程方面
研究主要包括时间敏感网络仿真与实物平台研制。其中，本人提出的仿真中间件框架可以实现网络协议仿真、研发、测试等环节代码一致性问题，并提供了协议故障注入功能，方便用户对协议进行加错测试。该项工程研究成果已经被VTC2025接收。

### 理论方面
研究包括分布式时间同步算法的开发与优化，旨在解决目前主从式时间同步方法在任务关键系统中可靠性低的问题，提高网络在不同故障场景下的容错能力；基于分布式容错原理提出了一套基于广播的容错时间同步算法(FTAS)。

此外，基于网络演算理论提出了一套实用网络演算算法，用于关键任务系统承载网络进行端到端延时的评估。针对强确定性业务，提出了一种时空资源分配算法，解决了这些业务路径与时隙分配问题。
