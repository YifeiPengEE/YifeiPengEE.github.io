Research Focus

Currently focused on researching and solving key problems in the theory and application of deterministic networking technologies.

Engineering Aspects

Research primarily involves Time-Sensitive Networking (TSN) simulation and physical platform development. I have proposed a simulation middleware framework that addresses code consistency issues across network protocol simulation, development, and testing phases, while also providing protocol fault injection capabilities for error testing. This engineering research achievement has been accepted by VTC2025.

Theoretical Aspects

Research includes the development and optimization of distributed time synchronization algorithms, aiming to solve the low reliability issue of master-slave time synchronization methods in mission-critical systems, thereby improving network fault tolerance in various failure scenarios. Based on distributed fault-tolerant principles, I have proposed a broadcast-based Fault-Tolerant Time Synchronization Algorithm (FTAS).

Additionally, based on network calculus theory, I have developed a practical network calculus algorithm for evaluating end-to-end delays in mission-critical system networks. For strongly deterministic services, I have proposed a spatiotemporal resource allocation algorithm that solves the path and time slot allocation problems for these services.
